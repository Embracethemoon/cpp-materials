## C++ 学习路线:

- 学习基本**语法**
	- 变量，表达式，输入，输出，循环，逻辑，结构体，函数等基本内容
- **STL**学习(Standard Template Library)
	- 一个常用的C++标准模板库(不需要理解实现原理，只用掌握部分基本用法)
- 考察语言掌握的**练习题**:
	- 一些可选的在线题目，用来巩固对语法的掌握
